# 🎨 Smart Campus Frontend

A modern, responsive user interface for the Smart Campus platform, built with React 19, TypeScript, and Vite.

## 🛠️ Tech Stack
- **Framework**: React 19 (TypeScript)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **State Management**: React Hooks & Context API
- **Routing**: React Router DOM

## ✨ Features
- **Interactive Dashboard**: Tailored views for each user role.
- **AI Chatbot Interface**: Sleek CampusBot widget for real-time assistance.
- **Event Management**: Visual tools for browsing and registering for events.
- **Responsive Design**: Seamless experience across all devices.

## 🚀 Getting Started
1. `npm install`
2. `npm run dev`
